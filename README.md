# stolov
Автоматизированная информационная система для учёта и контроля питания в
школьной столовой. Позволяет: авторизоваться учащимся, просматривать меню завтраков
и обедов, оплачивать питание (разово или абонементом), указывать
аллергены и другие пищевые особенности, оставлять отзывы; предоставлять сотрудникам столовой интерфейс для учета
выданных блюд, контроля остатков продуктов и оформления заявок
на закупки; обеспечивать администраторам доступ к статистике оплат и
посещаемости, согласованию заявок на закупку и формированию
отчетности <br>
<br>
Инструкция по установке:
1. Загрузить библиотеку "Flask" в терминале командой "pip install flask"
2. Создать venv в терминале комнадой "python -m venv venv"
3. Активировать venv командой в терминале "venv\Scripts\activate"
4. Установить требуемые библиотеки командой в терминале "pip install -r requirements.txt"
