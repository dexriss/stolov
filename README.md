# stolov
Автоматизированная информационная система для учёта и контроля питания в
школьной столовой. Предоставляет: регистрацию и авторизацию; просмотр меню завтраков и обедов; оплачивать питания (разовый платеж или абонемент); отмечать о получении питания;
указывать пищевые аллергии и предпочтения; оставлять отзывы о блюдах
